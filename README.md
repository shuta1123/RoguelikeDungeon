# Abyssal Echoes

> PCデスクトップ向けローグライクRPG — 深淵は囁く

## 概要

ランダム生成ダンジョンを探索し、モンスターを倒して最深部のボスを目指すローグライクRPG。
すべてのゲームロジックはGoバックエンドが処理し、フロントエンドはその結果を受け取って表示します。

## 技術スタック

| 役割 | 技術 |
|------|------|
| デスクトップ化 | Electron 28 |
| UI | React 18 + TypeScript |
| ビルド | Vite |
| 状態管理 | Zustand |
| API通信 | Axios（現在はモック） |
| スタイル | Tailwind CSS |

## ディレクトリ構成

```
├── electron/          # Electronメイン・プリロードプロセス
└── src/
    ├── api/           # API通信層（現在はモックを返す）
    ├── components/    # 再利用UIコンポーネント
    ├── mocks/         # 静的モックデータ（API代替）
    ├── screens/       # 各画面
    ├── stores/        # Zustand状態管理
    └── types/         # TypeScript型定義（APIレスポンス型）
```

## セットアップ

```bash
npm install
npm run dev
```

Electronウィンドウが起動します。

## 操作方法

| キー | 操作 |
|------|------|
| `W` / `↑` | 上へ移動 |
| `S` / `↓` | 下へ移動 |
| `A` / `←` | 左へ移動 |
| `D` / `→` | 右へ移動 |

モンスターに接触すると自動で戦闘画面へ遷移します。
